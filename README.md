# orlando-ai

AI-помощник владельца Chevrolet Orlando: RAG по чату сообщества + личная сервисная книжка + диагност по симптому и звуку. PWA, живёт на orlando-ai.vercel.app.

## Что умеет

- **Спросить** — ответ по 65K тредов чата сообщества (гибридный поиск: pgvector + full-text) с учётом истории именно этой машины.
- **Суммировать находки** — AI-сводка только по обсуждениям сообщества.
- **Спросить в интернете** — Perplexity Sonar с источниками.
- **🩺 Диагност** — триаж: описываешь симптом и/или записываешь звук с микрофона (до 15 сек) → ранжированные причины с уверенностью, светофор срочности (🟢/🟡/🔴), что проверить самому, опыт владельцев из чата. Честно говорит «данных мало» и задаёт вопросы вместо угадывания. Сервисная книжка работает фильтром вероятностей: недавно менянный узел — ниже вероятность, просроченный по регламенту — выше.
- **Сервисная книжка** — записи о работах свободным текстом (LLM разбирает в структуру), правка/удаление под PIN, экспорт CSV/JSON.
- **PWA** — установка на телефон, шорткаты (Диагностика / Записать работу / Спросить), офлайн-страница.

## Стек

Next.js 14 · Supabase (pgvector + FTS) · Mistral embed · OpenRouter (DeepSeek — ответы и триаж, GPT-4o-mini — парсинг записей, Perplexity Sonar — интернет, Gemini 2.5 Flash — анализ звука).

## Запуск

```powershell
npm install
copy .env.example .env.local   # и заполнить ключи
npm run dev                    # localhost:3000
```

Деплой: git push → Vercel. Правила проекта и грабли локальной разработки — в `CLAUDE.md`.

## Пайплайн данных сообщества (разовый)

```powershell
npm run parse     # JSON-экспорт Telegram → data/messages.jsonl
npm run threads   # → треды
npm run index     # эмбеддинги → Supabase
npm run stats     # статистика
```

`data/` в .gitignore — сырые выгрузки в репо не попадают.
